## switch语句

#### 语法

```c++
switch(表达式)
{
    case 值1:
        语句1;
        break;
    case 值2:
        语句2;
        break;
    ......
    default:
        上述条件都不满足时执行的语句;
}
```

* case后面必须是整数和字符，或者是结果为整数和字符的表达式，但不能使用变量
* default不是必须的
* 分支语句结束后不写break会走到下一个分支，直到执行剩余全部分支或者遇到break

## while语句

#### 语法

```c++
while(表达式)
{
    语句块;
}
```

当表达式为真时执行语句块，执行完成后重新判断，直到表达式值为假跳出循环

## 关系运算
用于比较两个表达式的值，结果为1-true和0-false
### 关系运算
|关系|数学中的表示|c++的表示|
|---|---|---|
|等于|=|==|
|不等于|≠|!=|
|小于|<|<|
|小于等于|≤|<=|
|大于|>|>|
|大于等于|≥|>=|

例子
```c++
#include <iostream>
using namespace std;
int main() {
	int a = 0;
	cout << "a=" << a << endl
		<< "a==0 is " << (a == 0) << endl
		<< "a!=0 is " << (a != 0) << endl;
	return 0;
}
/*-------------------------------------------------------
输出：
a=0
a==0 is 1
a!=0 is 0
*/
```
## 逻辑运算
根据给定的逻辑因子（表达式或值）返回一个新的逻辑因子
|运算符|术语|示例|结果|
|---|---|---|---|
|&&|逻辑与|a && b|如果a和b都为真则结果为真，否则为假|
|\|\||逻辑或|a \|\| b|如果a和b都为假则结果为假，否则为真|
|!|逻辑非（反）|!a|a为假则结果为真，反之为假|

* 逻辑运算符的两边可以是数值，也可以是表达式
* 用std::cout输出表达式时要加括号
* 实际开发中逻辑运算的多重组合是重点
